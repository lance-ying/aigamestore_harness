Pilot Barry's jetpack through a dangerous, endless lab. Fly as far as possible, collecting coins, avoiding obstacles, and using vehicle tokens for protection.

SPACE or ARROW UP: Activate Jetpack (Fly Up) / Flap Wings (Dragon Mode) | ESC: Pause / Unpause Game | ENTER: Start Game | R: Restart Game (on Game Over screen) | Z: Activate special vehicle ability (if equipped)
